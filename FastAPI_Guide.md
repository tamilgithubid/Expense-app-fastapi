# FastAPI Complete Setup Guide

A beginner-friendly guide to install Python, set up FastAPI, and build a Math API.

---

## 1. Install Python

### macOS

```bash
# Install Homebrew (if not installed)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install Python
brew install python
```

### Windows

1. Download Python from [python.org](https://www.python.org/downloads/)
2. Run the installer
3. **Check** "Add Python to PATH" during installation
4. Click "Install Now"

### Verify Installation

```bash
python3 --version
# Output: Python 3.x.x
```

---

## 2. Set Up Project

### Create Project Folder

```bash
mkdir fastapi-math-app
cd fastapi-math-app
```

### Create Virtual Environment

```bash
# Create
python3 -m venv venv

# Activate (macOS/Linux)
source venv/bin/activate

# Activate (Windows)
venv\Scripts\activate
```

> After activation, you'll see `(venv)` at the beginning of your terminal prompt.

### Install FastAPI & Uvicorn

```bash
pip install "fastapi[standard]"
```

This installs:
- **FastAPI** — the web framework
- **Uvicorn** — the ASGI server to run your app
- **Pydantic** — data validation
- **httpx**, **jinja2**, and other useful extras

### Verify Installation

```bash
python3 -c "import fastapi; print(fastapi.__version__)"
uvicorn --version
```

## 4. Create the Math API

Create a file named `main.py`:

```python
from fastapi import FastAPI, Query

app = FastAPI(
    title="Math API",
    description="A simple API that performs basic math operations",
)


@app.get("/")
def home():
    return {"message": "Welcome to the Math API!"}


# ---- Addition ----
@app.get("/add")
def add(a: float = Query(..., description="First number"),
        b: float = Query(..., description="Second number")):
    return {"operation": "addition", "a": a, "b": b, "result": a + b}


# ---- Subtraction ----
@app.get("/subtract")
def subtract(a: float = Query(..., description="First number"),
             b: float = Query(..., description="Second number")):
    return {"operation": "subtraction", "a": a, "b": b, "result": a - b}


# ---- Multiplication ----
@app.get("/multiply")
def multiply(a: float = Query(..., description="First number"),
             b: float = Query(..., description="Second number")):
    return {"operation": "multiplication", "a": a, "b": b, "result": a * b}


# ---- Division ----
@app.get("/divide")
def divide(a: float = Query(..., description="First number"),
           b: float = Query(..., description="Second number")):
    if b == 0:
        return {"error": "Cannot divide by zero"}
    return {"operation": "division", "a": a, "b": b, "result": a / b}


# ---- Power ----
@app.get("/power")
def power(base: float = Query(..., description="Base number"),
          exp: float = Query(..., description="Exponent")):
    return {"operation": "power", "base": base, "exp": exp, "result": base ** exp}


# ---- Square Root ----
@app.get("/sqrt")
def sqrt(n: float = Query(..., description="Number")):
    if n < 0:
        return {"error": "Cannot compute square root of a negative number"}
    return {"operation": "square_root", "number": n, "result": n ** 0.5}
```

---

## 5. Run the Server

```bash
# Make sure venv is activated
source venv/bin/activate

# Start the server
uvicorn main:app --reload
```

You will see:

```
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Started reloader process
```

## 9. Useful Commands

| Command | Description |
|---------|-------------|
| `source venv/bin/activate` | Activate virtual environment |
| `deactivate` | Deactivate virtual environment |
| `uvicorn main:app --reload` | Start server with auto-reload |
| `uvicorn main:app --host 0.0.0.0 --port 8080` | Run on custom host/port |
| `pip freeze > requirements.txt` | Save installed packages |
| `pip install -r requirements.txt` | Install from requirements file |
